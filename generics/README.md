## 泛型

- 泛型: 提高代码复用能力

  - 处理重复代码的问题

- 泛型是具体类型或其它属性的抽象代替

  - 你编写的代码不是最终的代码，而是一种模板， 里面有一些‘占位符’
  - 编译器在编译时将‘占位符’替换为具体的类型

- 例如: `fn largest<T>(list: &[T]) -> T { }`
- 类型参数:
  - 很短，通常是一个字母
  - T: type 的缩写
